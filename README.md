# 🦞 TurboClaw

**TurboClaw** 是一款强大且直观的本地 Agent 客户端应用，为您在本地管理和使用 AI 助手（Agents）提供了一站式、沉浸式的交互体验。无论是管理多模态模型、操作本地文件，还是将 AI 服务连接至各大聊天软件，它都能极速搞定。

> ⚠️ **项目定位说明**：本仓库主要用于发布 **macOS 端可执行程序** 与使用说明。您无需自行编译代码，直接在 [Releases](#) 页面下载最新版即可使用。

---

## ✨ 核心特性

- **🤖 本地驱动与完全掌控**
  - 在你的电脑上安全运行，无需过度依赖中心化云端服务。
  - 支持 **独特个性和长期记忆、主动性心跳机制、定时任务**。
  - 具备 **本地文件访问、编辑、整理、系统级命令行权限**。
  - **原生权限管控**：提供本地文件系统的可视化授权（`Desktop` / `Documents` / `Downloads`），保障隐私。
  
- **💬 沉浸式多会话聊天**
  - **多会话管理**：支持侧边栏随时新建、切换、删除不同的聊天上下文。
  - **上下文富媒体支持**：自动预读你的消息中包含的文件路径或「桌面」「文档」等关键词，并智能带入给 AI 识别。
  - **Markdown & 代码高亮**：原生支持富文本、代码高亮渲染以及一键复制。
  
- **🔗 一键接入各大聊天平台 (Bot 通道)**
  - 内置支持零代码接入 **Telegram, Discord, 飞书, 钉钉, QQ** 等平台。
  - 你只需要在设置页填入 Token 或 App ID 即可开启随身远程控制模式！
  - *甚至包含 @BotFather 设置提示指南，新手也能轻松驾驭。*

- **⚙️ 灵活的模型引擎配置**
  - 支持 **Zhipu (智谱)、OpenAI、Anthropic、DeepSeek、OpenRouter** 等主流模型供应商接入。
  - 默认内置 `glm-4.7-flash` 免配体验。
  - 原生无缝融合 [PicoClaw](#) 引擎。

## 📥 安装指南 (macOS)

1. 前往 **[Releases](#)** 页面，下载最新的 `TurboClaw_x.x.x.zip`。
2. 双击解压 ZIP 文件，您将得到 `TurboClaw.app`。
3. 将 `TurboClaw.app` 拖入您的 **「应用程序 (Applications)」** 文件夹。
4. **首次打开注意**：
   - macOS 可能会提示“无法验证开发者”，请**右键点击 app** ➡️ 选择**「打开」**；
   - 或者是进入 `系统设置` 👉 `隐私与安全性`，在底部点击 **“仍然打开”** 即可。

## 🚀 快速上手

1. 启动 TurboClaw 后，会首先提示授权本地目录访问权限。
2. 左下角点击 **Settings (设置)**，前往 **通道 (Channels)** 或 **模型 (Model)** 配置你想要的机器人和 API Key。
3. 你的所有配置将立刻生效，点击新建聊天，即可与 TurboClaw 开启对话！

## 🌍 国际化支持

App 原生支持以下语言，可在 `Settings` 中一键切换：
- 🇨🇳 简体中文
- 🇺🇸 English
- 🇪🇸 Español (Spanish)
